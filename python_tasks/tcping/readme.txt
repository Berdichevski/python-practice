TCPing — это утилита для проверки доступности TCP-порта на заданном хосте.
Она выполняет несколько попыток подключения к указанному хосту и порту,
замеряет время отклика и выводит статистику.

Программа выполняет TCP-пинг указанного хоста и порта,
измеряя время отклика для каждой попытки подключения.
Она также выводит информацию о количестве успешных и неудачных попыток,
минимальное, максимальное и среднее время отклика.

Автор: Бердичевский Артём

Запуск программы осуществляется из командной строки.
Программа поддерживает следующие аргументы:
python tcping.py <host> <port> [--count <attempts>] [--timeout <seconds>] [--interval <seconds>]

Пример использования:
python tcping.py example.com 80 --count 5 --timeout 3 --interval 2

Этот пример выполнит 5 попыток подключения к порту 80 на example.com,
с временем ожидания 3 секунды и интервалом между попытками 2 секунды.
