Название задачи:
Hex Editor

Описание задачи:
Это программа для редактирования двоичных файлов в формате HEX. Она позволяет пользователям просматривать содержимое файла, редактировать отдельные байты, искать данные по HEX или ASCII, а также сохранять изменения в исходный файл.

Примеры использования:
Открытие файла:
python hexeditor.py myfile.bin
Просмотр содержимого файла с заданного смещения:
view 000010
Переход к следующей странице:
next
Редактирование байта по смещению:
edit
Поиск данных в формате HEX:
search
Сохранение изменений:
save

Описание функционала:

view [offset] — Показывает содержимое файла с указанного смещения в шестнадцатеричном формате.

next — Переходит к следующей странице файла (по умолчанию 256 байт).

prev — Переходит к предыдущей странице файла.

goto [offset] — Переходит к конкретному смещению в файле.

edit — Редактирует байт по указанному смещению.

search — Осуществляет поиск данных в формате HEX или ASCII.

save — Сохраняет изменения в файл.

help — Показывает справку по доступным командам.

quit — Выход из программы, с возможностью сохранить изменения перед выходом.
